1. What is Sirius? 

Sirius in a nutshell

  • A tool to quickly define custom multi-view worbenches (based on graphical editors)

  • Based on & compatible with Eclipse technologies (EMF, EMFT, GEF/GMF, Acceleo, SWT/JFace)

What benefits?

  • For developers: dramatically reduces the cost & complexity of creating modeling worbenches

  • For end-users: stop adapting your work to the tools, adapt the tools to your work

Creating a Modeling workbench with Sirius

  • Specification environment (Specifier/Developer)
    • Define custom multi-view worbenches (Diagrams, Trees, Tables Editors...)
    • With little technical knowledge
    • Get a working modeler fast (POC)
    • Highly customizable (native tooling, Java or extension points)

  • The associated runtime (End-user)

Using a Sirius Modeling workbench

  • Specification environment (Specifier/Developer)
  • The associated runtime (End-user)
    • Work with data (models) - not just for doc
    • Environment adapted to user needs (Viewpoint-based)
    • Simplified workflow (Modeling Project)
